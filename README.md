## Setup

Required packages:
* NodeJS. Version 4!
* Python 2
* JDK 8
* React Native

More information about the required packages described [here](https://facebook.github.io/react-native/docs/getting-started.html).   

## Usage

Navigate to the `App` folder and run the following commands:    
`npm install`  
`npm start`
